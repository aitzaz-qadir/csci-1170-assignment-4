The Force: Weather Info

What is this web page all about?

Well, this web page shows a random weather value and based on the current temperature value somewhere in the universe, it will pick one of five images to show below the temperature as the "mood" image. The temperature could be anywhere between freezing to super hot, and that's what each image "mood" will try to visualize based on the temperature value.
